# Graphics Viewer

Description:

    Draw lines or triangles on a 3D world 
    Export custom defined Image files 
    Import stl/Image files for viewing 

How to use:

    make
    ./graphics


Controls:

    l  -  draw line 
    t  -  draw triangle
    w  -  vertical orbit up 
    s  -  veritcal orbit down
    a  -  horizontal orbit left
    d  -  horizontal orbit right
    e  -  rotate clockwise
    q  -  rotate counter clockwise
    i  -  import file (.stl or .txt with custom Image format)
    o  -  save file as custom Image format
    <  -  translate model left  (relative to view) 
    >  -  translate model right (relative to view) 
    ^  -  translate model up    (relative to view) 
    v  -  translate model down  (relative to view) 
    c  -  clear image on screen 
    n  -  new viewing window 
    ;  -  previous window
    '  -  next window
    space  -  reset view to default
    scroll up  -  zoom out FOV
    scroll down  -  zoom in FOV
